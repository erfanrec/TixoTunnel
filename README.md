# TixoTunnel

Minimal tunnel management console powered by **AETHER-X1**.

**NOVA-5.0** supports TUN, IPX, encryption, auto tuning, multi-tunnel management, live monitoring, and optional IP spoofing/custom packets.

## Install

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/erfanrec/TixoTunnel/main/TixoTunnel.sh)
```

Run the panel later with:

```bash
tixotunnel
```

## Repository

Keep these two files in the repository root:

```text
TixoTunnel.sh
README.md
```

Upload the engine to the latest GitHub Release with this exact asset name:

```text
tixotunnel-core
```
